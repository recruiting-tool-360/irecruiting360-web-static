# AI Job （node：18.20.3）

- api:接口文档
- assets：图标和照片文档
- components：组件库
- layout：布局文档
- router: 路由文件
- store：存储文件
- views：页面和内部文件

```
npm install
```

npm run serve

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
